## Setup Instructions

1. **Place all three folders in the same directory**
   - Keep Homepage, English, and History folders at the same level
   - Do NOT move or rename the folders

2. **Open the website**
   - Navigate to the Homepage folder
   - Open `index.html` in your web browser
   - This is your starting page!

## Troubleshooting

**Problem:** Navigation links don't work
**Solution:** Make sure all three folders are in the same directory

**Problem:** English quotes don't show
**Solution:** Make sure quotes.js is in the English folder

**Problem:** Sidebar doesn't open
**Solution:** Check that the JavaScript files are in their respective folders
